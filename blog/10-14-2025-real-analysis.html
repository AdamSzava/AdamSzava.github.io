An exercise about the supremum and set inclusion.

Posted on: 10-14-2025

Proposition: Let $A \subseteq B \subseteq \mathbb{R}$. Then $\sup(A) \leq \sup(B)$.

Proof:

Suppose not, as in suppose that $A \subseteq B$ and that $\sup(A) > \sup(B)$. Then we can find an element of $A$ which is not in $B$ (contradicting that they are subsets).

Let $\varepsilon = \sup(A)- \sup(B) > 0$, then by definition of the supremum, we have that $$\exists a \in A \text{ s.t. } \sup(A)-\varepsilon < a \leq \sup(A)$$

Notably, since $\sup(A)- \varepsilon = \sup(B)$ we have that $\sup(B) < a$. $\sup(B)$ is an upper bound for $B$, so all of its elements are at most $\sup(B)$. Since $a$ is strictly bigger we have that $a \notin B$. This contradicts the fact that $A \subseteq B$.

$\blacksquare$
