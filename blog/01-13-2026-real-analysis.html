The Interior Extremum Theorem

Posted on: 01-13-2026

We follow Understanding Analysis by Stephen Abbott (2e) for the proof of the following.
Theorem 5.2.6 (Interior Extremum Theorem): Let $f$ be differentiable on $(a,b)$ and let there be some $c\in(a,b)$ for which $c$ is a maximum in the sense that $f(c) \geq f(x) \forall x\in (a,b)$. Then $f'(c) = 0$.

Proof:

We aim to show $f'(c) = 0$. Let $d(x)$ denote the difference quotient at $c$, as in $$d(x) = \frac{f(x)-f(c)}{x-c}$$Note that we know the limit $\lim_{x \to c} d(x)$ exists and equals $f'(c)$. The idea is this: Recall the sequential characterization of the limit which states that if we have an arbitrary sequence $z_n \to c$, then $\lim_n d(z_n) = f'(c)$. If $d(z_n) \geq 0 \forall n\in \mathbb{N}$, then the order limit theorem would tell us that $\lim_n d(z_n) = f'(c) \geq 0$ as well. Similarly if $d(z_n) \leq 0 \forall n\in\mathbb{N}$ we'd have $f'(c) \leq 0$.

We construct two such sequences. Note that since $c\in (a,b)$ we can split this domain into $(a,c] \cup [c,b)$ and define a sequence $x_n \to c$ on the smaller set, and a sequence $y_n \to c$ on the greater one.

Indeed, for all $n\in \mathbb{N}$ let $x_n := c- \frac{c-a}{n+1}$, and $y_n := c+ \frac{b-c}{n+1}$. Clearly we have:
  • $x_n \to c$, and $y_n \to c$
  • $a < x_n < c < y_n< b \forall n\in \mathbb{N}$
  • As already mentioned since $\lim_{x\to c} d(x)$ exists we also have that $\lim_n d(x_n) = \lim_n d(y_n) = f'(c)$.
Now we claim that $d(x_n) \geq 0$ and $d(y_n) \leq 0 \forall n\in \mathbb{N}$ and so by the order limit theorem we have that $\lim_nd(x_n) = f'(c) \geq 0$ and $\lim_nd(y_n) = f'(c) \leq 0$, and so $f'(c) = 0$.

Indeed, consider the following:

$$d(x_n) = \frac{f(x_n)-f(c)}{x_n-c} \geq 0$$

$$d(y_n) = \frac{f(y_n)-f(c)}{y_n-c} \leq 0$$

Since in the first one both the numerator and denominator are negative always, and in the second one the numerator is negative while the denominator is positive always.

$\blacksquare$

Note that this extends to local maxima as well. By definition a local maximum is a point $c$ in the domain such that there exists an open interval $(a',b') \subseteq (a,b)$ such that when the function is restricted to that domain, the value $f(c)$ becomes the maximum. Then just apply the same theorem.
